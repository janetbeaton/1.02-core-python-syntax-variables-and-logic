## 1. Python Syntax and Data Types

Python uses a clean, readable syntax designed to make code easy to understand.

Common data types include:

- Integers (`int`)
- Floating-point numbers (`float`)
- Strings (`str`)
- Booleans (`bool`)

Example:

```python
age = 30
height = 1.65
name = "Rita"
is_student = True
```
---

## 2. Variables and Expressions

Variables store values that can be reused and updated throughout your program.

```python
x = 10
y = 5
total = x + y
total
```
Clear variable naming improves readability and reduces errors.

---
## 3. Control Flow with Conditionals

Conditionals allow your program to make decisions based on logic.

```python
score = 72

if score >= 50:
    print("Pass")
else:
    print("Fail")
```
Comparison operators include:

- `==`, `!=`  
- `>`, `<`, `>=`, `<=`

---

## 4. Control Flow with Loops

Loops automate repetitive tasks.

### For Loops

```python
for i in range(5):
    print(i)
```
### While Loops

```python
count = 0
while count < 3:
    print(count)
    count += 1
```
Loops are essential for processing data and automating workflows.

---

## 5. Versioning Python Scripts with Git

Version control helps track changes and supports reproducibility.

```bash
git add .
git commit -m "Add core Python syntax and logic examples"
```
Using GitHub allows you to share and review your code professionally.
